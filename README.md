# BICA PDF → Excel Converter

Convierte extractos bancarios del Banco BICA SA a Excel con columna
"Cuenta Contable" lista para clasificar.

## Instalación (una sola vez)

Requiere Python 3.8+.

```
pip install flask pdfplumber openpyxl
```

## Uso

Doble clic en `iniciar.bat` (Windows) — o desde la terminal:

```
python app.py
```

Se abre el navegador en http://localhost:5050
Arrastrá el PDF → hacé click en "Convertir" → descarga automática.

## Cerrar

Cerrá la ventana de la terminal (o Ctrl+C).
